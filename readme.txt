Customizes the WordPress admin menu with dynamic submenu alignment and arrow colors based on the admin color scheme.

== Description ==
This plugin enhances the WordPress admin menu:
- Submenu alignment adjusts dynamically to available screen space.
- Arrows match the user's selected admin color scheme for a seamless UI.

== Installation ==
1. Download the plugin from [GitHub](https://github.com/Lyhtande/Wordpress-Custom-Admin-Menu-Plugin).
2. Upload the ZIP file via **Plugins > Add New > Upload Plugin** in your WordPress dashboard.
3. Activate the plugin to apply enhancements to your admin menu.
